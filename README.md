# FunVault 🎮

FunVault is a simple website where users can play small and fun browser games.  
All games are built using basic web technologies like HTML, CSS, and JavaScript.

## 🚀 About the Project
This project is a collection of simple games that run directly in the browser.  
The goal of FunVault is to provide quick and fun games without needing to download anything.

## 🕹️ Games
Some games currently available in FunVault:

- Rock Paper Scissors
- Hand Cricket
- More games coming soon...

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 📌 Purpose
This project is made for learning and practicing web development and JavaScript game logic.

## 🔮 Future Plans
- Add more browser games
- Improve UI and animations
- Add player vs player modes
- Make the website more interactive

## 📂 How to Use
1. Open the website.
2. Choose a game.
3. Start playing!
